# Документация по блокам - play_effect

`play_effect` - это реакция на событие, которое порождает эффект частиц относительно целевой позиции.

## Дополнительные параметры

`play_effect` может использовать следующие параметры

| Имя    | Значение по умолчанию | Тип              | Описание                         |
|--------|-----------------------|------------------|----------------------------------|
| data   | 0                     | Integer          | Значение данных частицы.         |
| effect | not set               | String           | Имя создаваемого эффекта частиц. |
| target | self                  | Minecraft Filter | Целевой контекст для выполнения. |

## Пример

``` json
"play_effect":{
    "delta" : 0,
    "effect" : "particle_sparks",
    "target" : {"test":    "effect" : "particle_sparks",
    "target" : { "test":  "random_chance", "value": "3"} //25%
}
```
